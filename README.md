🔐 Gerador de Senhas Premium

Um gerador de senhas desenvolvido a partir do projeto da Alura, personalizado com uma identidade visual elegante em tons de vermelho e dourado.

📖 Sobre o Projeto

Este projeto permite gerar senhas aleatórias de forma rápida e segura, oferecendo opções de personalização para aumentar a proteção das credenciais dos usuários.

O visual foi reformulado para apresentar um design sofisticado, utilizando uma combinação de vermelho escuro, dourado e preto, inspirada em sistemas de segurança premium.

✨ Funcionalidades

- Geração automática de senhas aleatórias.
- Definição da quantidade de caracteres.
- Inclusão de letras maiúsculas.
- Inclusão de letras minúsculas.
- Inclusão de números.
- Inclusão de símbolos especiais.
- Indicador visual de força da senha.
- Estimativa de tempo para quebra da senha baseada em entropia.
- Interface moderna e responsiva.

🎨 Personalização

O projeto recebeu uma nova identidade visual com:

- Fonte elegante para títulos.
- Paleta de cores vermelha e dourada.
- Efeitos visuais modernos.
- Melhor contraste para acessibilidade.
- Aparência inspirada em cofres e sistemas de segurança digital.

🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

🚀 Como Executar

1. Faça o download ou clone este repositório.
2. Abra a pasta do projeto.
3. Execute o arquivo "index.html" em seu navegador.
